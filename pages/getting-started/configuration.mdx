# Configuration

| Name          | Type       | Default Value | Description                                            | Supported in CLI |
|---------------|------------|---------------|--------------------------------------------------------|-------------------|
| repoPath      | `string`     | `null`          | Absolute path of the repository or GitHub URL          | x |
| ignoreFolders | `string[]`   | `[]`            | List of folder names to ignore during the scan          |
| useDefaultPlugins | `boolean`   | `true`            | Whether to use the default built-in plugins or not          |
| plugins | `(Plugin\|string)[]`   | `DefaultPlugins`       | List of plugins to be used, can use the Plugin object or the full path          |

### DefaultPlugins

The default plugins are: TSFilePlugin, ImportsPlugin, ConsoleLogFinalizer, UsedHTMLTags, ChartJSPlugin

For more information about these plugins, check out the [built in plugins](../plugins/built-in) page

### Plugin

| Name          | Type       | Default Value | Description                                            |
|---------------|------------|---------------|--------------------------------------------------------|
| disabled      | `boolean`     | `false`          | Disable the plugin          |
| params | `any`   | `null`            | Custom parameters to pass the plugin          |
| path       | `string`   | `null`         | An absolute path to the plugin, or plugin name if using one of the built-in ones |


## Configuration Example

```ts
// analyzer.config.json
import { ConfigModel } from '@codeinsights/js';

const config: ConfigModel = {
  ignoreFolders: ['node_modules', 'public', 'src/assets'],
  plugins: [
    'angular',
    {
      path: 'plugins/used-html-tags.ts',
      params: {
        matchPattern: '^mat' // only components from angular material,
      },
    },
  ],
};
  disabled?: boolean;
  params?: T;
  path: string;
  beforeHooks: Record<BeforeHookKeys, string>;

module.exports = config;
```
