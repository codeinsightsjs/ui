# OpenSearch Plugin

## Plugin Details

The OpenSearch Plugin allows you to send your logs to OpenSearch. You can then use OpenSearch to explore
the logs and Open Search Dashboards
to set up visualizations.

| id                 | Is Used By Default
|---------------|--------------------------------------------------------
| opensearch    |          |

## Parameters

| name                 | Description | Default Value
|---------------|------------------------------ | ------
| indexPattern    | OpenSearch index pattern          |  analyzer_index |
