# CLI

```bash
$ npx @codeinsights/js@latest --repoPath "/path/to/repo"
```

## CLI Params

| Name          | Type       | Default Value | Description                                            |
|---------------|------------|---------------|--------------------------------------------------------|
| repoPath      | `string`     | `null`          | Absolute path of the repository or GitHub URL      |
| plugins      | `string`     | `null`          | List of plugins paths, separated by comma. e.g. `User/Desktop/custom-plugin.ts, opensearch`      |
